TEKNews

This is a SwiftUI News Application I made in Xcode. It derives the lastest headlines from various news publications (CNN,MSNBC,FOX) by country and displays that data in a Table View Format With images, and a locally hosted browser using the Webkit module.

This app was made as a submission to the Microsoft Hack For Africa Event but I've been refining and polishing it since. If I continue adding features and defining its purposes, I'll consider an app store upload.

The TikTok Bot

Welcome to the TikTok Bot! This bot is a tool for automating the share/view process on TikTok. It is to be used for educational purposes, only. The bot is made in Python and uses OOP and multi-processing to exploit the number of requests we can make to the TikTok webpage to iteratively add views/shares.

However, please keep in mind that while some parts of the code work, others are patched and will not be. If you come across any typos, issues, or documentation errors, please feel free to create a new issue inside the Bot's repo.

A* Pathfinding Visualization Tool

This is my approach towards the "cliche" algorithm visualization tool it seems almost everyone has as a side project these days. Source code is in Python and utilizes the Pygame module to make all the visuals and nodes. Features include colorful visuals, placeable nodes, unlimited barriers, and clean execution.
